BDB::XML::Document

A Document is the unit of storage within a Container.

A Document contains a stream of bytes that may be of type XML or BYTES. The Container only indexes the content of Documents that contain XML. The BYTES type is supported to allow the storage of arbitrary content along with XML content.

Document supports annotation attributes.
